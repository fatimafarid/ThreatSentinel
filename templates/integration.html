System Requirements

Before starting, make sure your system meets the following:

  • Operating System: Ubuntu (20.04 or higher)
  • Python: Version 3.8 or above
  • Wireshark: Installed for packet capture and log analysis
  • Flask Framework: Installed to run the local service endpoint
  • Internet Access: To pull IOCs and download reports securely
1

Step 1: Install Required Tools

Run the following commands in your terminal:


                        # Update System
                       sudo apt update && sudo apt upgrade -y

                       # Install Wireshark
                      sudo apt install wireshark -y

                      # Install Python & pip
                      sudo apt install python3 python3-pip -y

                     # Install Flask and Dependencies
                    pip3 install flask requests
2

Step 2: Clone or Download Integration Scripts

Clone our integration tools from the repository:


                            git clone https://github.com/threatsentinel/script-integration.git

                            cd threatsentinel-integration

Key Scripts: pull_ioc.py (IOCs), report.py (Reports), config.json (Settings)

3

Step 3: Configure Connection Settings

Open config.json and update it with your organization details:


{
  "organization_id": "ORG-001",
  "server_url": "http://central-flask-server:5000",
  "api_key": "your-issued-api-key"
}

‚ö†Ô∏è **Important**: Replace "your-issued-api-key" with the key provided to your organization.

4

Step 4: Run Flask Server Locally

This service enables bi-directional communication with the central threat intelligence server.


python3 app.py

Expected Output: * Running on http://127.0.0.1:5000

5

Step 5: Pull IOCs and Reports

Execute the scripts to manually pull data and reports (these can be scheduled later):


                        # Pull Latest IOCs
                        python3 pull_ioc.py

                        # Download Reports (ZIP of Rules)
                        python3 downloadreport.py
6

Step 6: Verify Integration

A successful setup means:

  • You should receive **IOC updates** periodically.
  • Downloaded `.zip` reports will be stored in the local **reports/** folder.
  • **Logs** will confirm successful communication with the Flask server.
